# 🎒 校外教學大考驗 (Field Trip Challenge)

這是一個專為國小學生設計的 **互動式地理位置闖關遊戲**。老師可以透過 Google 試算表輕鬆設計任務點，學生則透過手機或平板，在地圖導航的指引下進行校外教學探索與知識問答。

---

## 🚀 快速開始 (Quick Start)

現在趕快開始進行校外教學大考驗吧：

* **🛠️ 老師端：製作路線** [點此進入老師後台](https://你的GitHub帳號.github.io/專案名稱/admin.html)  
  *(功能：搜尋地點、設定題目、發布遊戲 ID)*

* **🎮 學生端：遊戲開始** [點此進入學生遊戲](https://你的GitHub帳號.github.io/專案名稱/game.html)  
  *(功能：自動導航、隨機任務挑戰、積分結算)*

> **小撇步**：學生進入遊戲後，若想先熟悉操作，可以直接點選「試玩模式」進入預設的 `text001` 挑戰！

---

## ✨ 核心功能

* **📍 自動導航**：系統會自動引導學生前往下一個目標點，抵達時自動彈出對話，無需手動尋找。
* **🎲 隨機任務**：任務點（Task）順序隨機打散，確保每位學生的冒險路徑不同，增加趣味性。
* **🛡️ 老師防呆**：設定「起點」與「終點」時，系統會自動優化輸入介面，防止誤填。
* **📈 即時計分**：系統自動根據任務數量均分 100 分，滿分完成挑戰超有成就感。
* **🏗️ 線性流程**：嚴格遵守「起點 ➔ 隨機任務 ➔ 終點」的邏輯，故事連貫不中斷。

---

## 🛠️ 系統架構與安裝

1. **Google Sheets & Apps Script**：作為後端資料庫，儲存所有關卡資訊。
2. **老師管理端 (`admin.html`)**：老師發布任務的控制台。
3. **學生遊戲端 (`game.html`)**：學生使用的移動端闖關介面。

### 安裝步驟：
1. **設定 GAS**：將 `Code.gs` 部署為網頁應用程式，取得 API 網址。
2. **填寫連結**：在 `admin.html` 與 `game.html` 的代碼中填入你的 `gasUrl`。
3. **發布網頁**：將檔案上傳至 GitHub 並開啟 **GitHub Pages** 功能。
